# DIY Guardian v3.0 🛡️

DIY Guardian ialah sebuah **Chrome Extension** berasaskan sistem *rule-based* yang dibina untuk menganalisis dan menilai keselamatan video DIY di YouTube secara masa nyata (*real-time*). Projek ini dibangunkan sebagai produk inovasi bagi **Projek Penyelidikan I/II (CFT3124), Fakulti Teknologi Kreatif dan Warisan (FTKW), Universiti Malaysia Kelantan**.

## Kelebihan Utama ✨
* **100% Percuma & Offline:** Berfungsi sepenuhnya tanpa memerlukan kunci API (API Key) atau sambungan pelayan luar (menjaga privasi).
* **Sistem Rule-Based Ringan:** Menggunakan Manifest v3, HTML, CSS, dan JavaScript tulen untuk memadankan teks metadata video dengan pangkalan data kata kunci keselamatan.
* **Analisis & Sokongan Interaktif:** Mengesan 6 kategori utama DIY (Elektrik, Paip, Automotif, Kayu, Konkrit, Cat) serta menyediakan amaran keselamatan, keperluan PPE, *checklist* langkah kerja, rating kefahaman, dan nota peribadi.

---

## Cara Pemasangan & Pengujian (Installation) 🛠️

Ikuti langkah mudah ini untuk memasang extension secara manual (Developer Mode):

1. **Muat Turun Projek:**
   Klik butang **Code** (berwarna hijau di atas) dan pilih **Download ZIP**. Ekstrak (*unzip*) fail tersebut menjadi satu folder biasa di komputer anda.

2. **Buka Pengurusan Extension Chrome:**
   Buka pelayar Google Chrome, taip `chrome://extensions/` pada ruangan URL, dan tekan *Enter*.

3. **Aktifkan Developer Mode:**
   On suis **Developer mode** di penjuru kanan atas paparan Chrome Extension.

4. **Tarik & Masuk (Load Unpacked):**
   Tarik (*drag and drop*) folder hasil ekstrak tadi terus ke dalam halaman Chrome Extension tersebut. *(Atau klik butang **Load unpacked** di atas kiri dan pilih folder projek).*

5. **Pin & Uji di YouTube:**
   * Klik ikon teka-teki (🧩) di bar atas Chrome dan **Pin (📌)** nama *DIY Guardian*.
   * Buka mana-mana video DIY di **YouTube** untuk melihat panel antarmuka (UI) berfungsi secara automatik. 
   * *Nota: Jika panel tidak muncul semasa bertukar video, sila klik **Refresh (🔄)** pada halaman YouTube tersebut.*
